수율 최적화 추천 Agent — 과제 구조 한눈에 보기  > 현장(물리) → 시스템(데이터) → Agent(분석·검증) 3층 구조

웨이퍼 테스트에서는 바늘이 칩에 닿는 품질(Contact)이 나쁘면 멀쩡한 칩도 불량(가성fail)으로 판정되어 Lot마다 수율이 출렁인다. 이 Agent는 현장의 3개 데이터 소스를 Lot ID로 연결해, 과거 수율 Top 10% Lot이 썼던 Contact 조건을 찾아내 신제품(S5E9965) 양산 전에 최적 세팅을 추천하고, 추천 Lot vs 기존 Lot의 수율 비교로 효과를 검증한다.
① 현장Wafer Test 공정
(물리 세계)
웨이퍼 Lot 입고 Lot = 웨이퍼 묶음 고객이 만든 S5E9965 웨이퍼가 Lot 단위로 입고. Lot이 생산·이력·수율 관리의 기본 단위
→
Prober + Probe Card가 칩에 접촉 Contact Chuck 위 웨이퍼에 Probe Card 바늘을 눌러 전기 연결. 이때의 세팅 = Contact 조건: Prober(Chuck) 평탄도 · P/Card 평탄도 · OD(바늘을 더 눌러주는 깊이, um) · Sanding(바늘 끝 연마 #400 사포·횟수)
→
Tester 전기 측정 Test 칩(die)마다 신호를 넣어 pass / fail 판정. 어떤 장비·호기가 언제 어떤 테스트를 했는지가 이력으로 남음
→
Lot 수율 산출 Yield 수율 = 양품 die ÷ 전체 die. Lot별로 집계되어 모니터링됨
⚠ Contact이 불량하면(평탄도 틀어짐·OD 부족·바늘 오염) 접촉 저항이 커져 정상 칩도 fail로 오판(가성fail) → 같은 제품인데 Lot 간 수율 편차 발생. 양산 전 신제품은 최적 Contact 조건을 모르는 상태라 엔지니어 경험으로 세팅해 왔다 — 이 경험을 데이터로 대체하는 것이 본 과제.

3개 소스를 한 Lot으로 잇는 연결고리
설비 Log → Contact 조건 Prober·장비가 남기는 설정/측정 로그. OD·Sanding·평탄도가 여기 있음. 장비·작업자 PC 단위로 흩어져 있어 위치 파악에 1~2주 (최난관)
MOS (MES) → Test 이력 생산실행시스템. 어떤 Lot이 어느 장비·호기에서 언제 어떤 테스트를 받았는지의 수행 이력
Remon → 수율 결과 Lot별 수율 집계·모니터링 시스템. 분석의 목표 변수(Y)인 수율이 여기서 나옴
Lot ID 기준 3종 결합 → "Lot 1줄 = Contact 조건 + Test 이력 + 수율" 통합 테이블

③ Agent분석 → 추천 → 검증
1. 수율 상위 Lot 추출 과거 Lot들을 수율로 정렬, Top 10% Lot만 선별
→
2. 공통 Contact 조건 분석 Top 10% Lot들이 공유한 조건 패턴을 LLM + Data Tree(조건↔수율 관계 구조, 엔지니어와 2일 구축)로 도출
→
3. 추천 조합 출력 Prober 평탄도 ±3um · P/Card ±5um · OD 45 · Sanding #400/3회 — "이 세팅으로 시작하라"
→
4. 양산 전 적용 신규 S5E9965 Lot의 초기 Contact 세팅으로 적용 (엔지니어 감 → 데이터 근거)
추천 조합으로 진행한 LotAgent가 제시한 세팅으로 테스트
VS
기존 방식 Lot엔지니어 경험 세팅으로 테스트
→
Lot 수율 비교로 효과성 검증 두 그룹의 수율 차이 확인 (실 Lot 비교 불가 시, 과거 실적상 적용 가능한 조합인지로 대체 검증) — 현업 엔지니어 3명 × 1주 검수
양산 (Mass Production)제품 검증을 마치고 대량 생산·테스트에 들어가는 단계. "양산 전" = 최적 조건이 아직 정해지지 않은 시점
Lot / 수율Lot = 함께 처리되는 웨이퍼 묶음(관리 단위). 수율 = Lot 내 양품 칩 비율 — 테스나 사업 성과의 핵심 지표
Contact 조건바늘이 칩에 닿는 품질을 결정하는 세팅 4종(평탄도 2종 · OD · Sanding). 접촉이 나쁘면 가성fail↑ → 수율↓
OD (Over Drive)바늘이 패드에 닿은 뒤 추가로 눌러주는 깊이(um). 부족하면 접촉 저항↑, 과하면 칩 손상 — 업계 연구상 ~45um에서 접촉 저항 안정
Sanding (니들 연마)연마 시트(#400 입도)로 바늘 끝 오염·이물을 닦는 클리닝. 횟수가 접촉 품질에 영향 (#400/3회 = 시트 입도·횟수)
