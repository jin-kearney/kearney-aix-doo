## Key Excerpts

**Overview:**
- AVEVA PI System (formerly OSIsoft PI System): "the industry leader in data management, specifically designed to conquer the challenges of industrial environments"
- "An impressive industrial data historian that can store decades of information — specifically designed to capture, store, and add context to data from industrial processes"

**Core Functionality (3 areas):**
1. Collect: "provides a rich set of options for gathering high-fidelity, time series data in real-time from diverse and complex industrial systems"
2. Store & Enhance: Data Archive is the core time series database / historian; stores real-time and historical process data
3. Deliver: "ensures integrity and availability of critical real-time data"

**PI System Components:**
- PI Data Archive: core time-series data store (historian)
- PI Asset Framework (PI AF): asset hierarchy and metadata management
- PI Vision: visualization layer
- PI Web API: RESTful interface for programmatic data access

**Integration Capabilities:**
- PI Web API: RESTful interface for direct historical data storage without intermediate steps
- PI Connectors (PI interfaces): collect data from PLCs, DCS, OPC servers
- Integrates with: SAP, SQL databases, cloud platforms

**AI/ML Data Prep Role:**
- PI System stores decades of time-series sensor data → primary source for training predictive maintenance models
- HighByte Intelligence Hub has PI System connection module for contextualization
- Seeq reads directly from PI System via AVEVA PI connector
- Cognite Data Fusion ingests from PI System via connectors

**Industry Adoption:**
- Widely deployed in: oil and gas, power generation, pharmaceuticals, chemicals, food and beverage, metals and mining
- More than 65% of Fortune 500 industrial companies use PI System (historically)
