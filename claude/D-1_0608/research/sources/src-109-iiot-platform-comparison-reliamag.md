## Key Excerpts (from search result)

**PTC ThingWorx:**
Deepest development platform for custom IIoT applications with model-driven, rapid-application-development environment. Strong ties to PTC's CAD and PLM products. Lets teams model assets, build mashups/dashboards, orchestrate workflows around data from machines and sensors. Pending sale to TPG private equity introduces transition uncertainty.

**Litmus Edge:**
Best vendor-neutral approach for plants running mixed equipment from multiple vendors — 250+ protocol support, no cloud lock-in. Software designed to collect, process, and analyze data from industrial devices and systems at the edge. Features: device management, data normalization, protocol translation, rapid application deployment.

**Siemens Insights Hub (formerly MindSphere):**
Most turnkey path for Siemens-equipped plants with native digital twin integration and OEE analytics. Supports deployment across public cloud and industrial edge.

**AWS IoT SiteWise:**
Most flexible pay-as-you-go model for plants already on AWS. VW uses AWS IoT Greengrass, IoT Core, IoT Analytics, and IoT SiteWise to collect, assimilate, and analyze data from plants.

**Selection Guide:**
- Siemens Insights Hub: for Siemens plants
- AWS IoT SiteWise: for AWS-native organizations
- Azure IoT: for Microsoft shops
- PTC ThingWorx: for custom development
- Rockwell FactoryTalk: for Allen-Bradley environments
- GE Vernova Proficy: for process industries
- Litmus Edge: for vendor-neutral connectivity

**Pricing:**
- Cloud platforms (AWS, Azure): pay-as-you-go, small plant $500–$5,000/month
- Enterprise platforms (Siemens, PTC, Rockwell): typically start at $50,000–$100,000+/year for mid-size deployments
