기존 RFP상에서 두산에서 구상한 Framework. 
## Findable

### 데이터 카탈로그
산재되어 있는 데이터를 한 곳에 모아 검색하고 탐색할 수 있게 해주는 데이터 통합 체계입니다.
### 메타데이터
"데이터에 대한 데이터(Data about Data)"로서 어떤 정보를 담고 있는지, 어떤 속성과 규칙을 갖는지 알려주는 데이터 설명서입니다.
### 비즈니스 Glossary
조직에서 사용하는 용어의 의미를 통일해 주는 표준 용어사전으로, 명확히 정의된 공통 언어 역할을 합니다.
## Understandable
### 데이터 전처리
AI 모델이 이해하기 적합한 형태로 데이터를 정제하고 변환합니다. 원천 데이터의 노이즈, 결측치, 불명확성 등을 일관된 구조로 변환하는 과정입니다.
### 데이터 해설/주석
데이터 학습을 위한 정답 혹은 핵심 내용을 요약하여, AI가 학습 및 추론할 수 있는 정보를 제공합니다. (데이터에 대한 포스트잇 역할)
### 온톨로지
도메인 지식의 개념과 관계를 구조화하여 지식을 체계적으로 정리하고, AI 모델이 맥락을 이해하고 추론할 수 있도록 돕습니다. (비즈니스 지식의 지도)
## Trustworthy
### Observability
데이터, 데이터 파이프라인 및 환경을 파악할 수 있도록 지원합니다. 현 상태와 이상 징후를 지속적으로 모니터링하여 데이터 운영 안정성을 보장합니다.
### 데이터 품질/권한관리
데이터 사용자 접근 권한 통제를 통해 규정과 정책에 맞도록 보안 및 컴플라이언스를 유지합니다.
### 데이터 계통 (Lineage)
데이터의 출처, 변환, 흐름을 추적하여 데이터의 전·후 맥락을 이해할 수 있도록 보장합니다. (End-to-end view of data flow)
## Governed
### 데이터 생명주기 관리
데이터의 생성부터 폐기까지 전 생애주기를 체계적으로 관리해 효율성과 보안을 확보하고, 비용·유실 및 법적·보안 리스크를 최소화합니다.
### 데이터 운영관리
데이터 공급 전 과정의 신속한 활용을 지원하며, DevOps 통합 및 코드 기반 관리를 통해 데이터 품질과 전사적 협업을 강화합니다.
### 데이터 Product화
데이터를 누구나 즉시 사용할 수 있는 상품으로 제공해 활용성과 재사용을 높이고, 데이터에 대한 명확한 책임과 서비스 수준으로 신뢰를 보장합니다.
