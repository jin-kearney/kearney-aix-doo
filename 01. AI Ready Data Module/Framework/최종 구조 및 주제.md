# Findable

### 데이터 카탈로그

전사 데이터 자산의 위치·소유자·활용 정보를 통합 관리하여 검색과 탐색을 지원하는 체계

### 메타데이터

데이터의 구조·속성·생성 이력·품질 기준 등을 설명하는 데이터 설명 정보 체계

### 비즈니스 Glossary

업무 용어와 데이터 정의를 표준화하여 조직 내 공통 이해를 지원하는 비즈니스 용어 체계

---

# Understandable

### 데이터 전처리

AI 및 분석 활용 목적에 맞춰 데이터를 정제·변환·표준화하는 데이터 준비 과정

### 데이터 해설/주석

데이터의 의미·맥락·정답 기준을 설명하거나 태깅하여 AI 학습과 해석 가능성을 높이는 체계

### 온톨로지

도메인 개념과 관계를 구조화하여 AI의 맥락 이해와 지식 기반 추론을 지원하는 체계

---

# Trustworthy

### Observability

데이터와 파이프라인의 상태·흐름·이상 징후를 모니터링하는 운영 가시성 체계

### 데이터 품질 권한관리

데이터 품질 기준과 사용자 접근 권한을 관리하여 신뢰성·보안·컴플라이언스를 확보하는 체계

### 데이터 계통 (Lineage)

데이터의 출처·변환·이동 경로와 변경 이력을 추적하는 데이터 흐름 관리 체계

---

# Actionable

### Physical 데이터

운영 시스템·설비·현장 환경 등에서 생성·수집되는 실제 원천 데이터 자산

### API/Tool 연계 데이터

업무 시스템, 외부 서비스, AI 도구와 연동되어 즉시 활용 가능한 인터페이스 기반 데이터

### Prompt/하네스 자산화

프롬프트, 테스트 하네스, 실행 템플릿을 재사용 가능한 AI 활용 자산으로 관리하는 체계


---

# Evolvable

### 데이터 Product화

데이터를 명확한 소유권·품질 기준·서비스 수준을 갖춘 재사용 가능한 상품형 자산으로 관리하는 방식

### 합성데이터 (Synthetic)

실제 데이터 특성을 기반으로 인공 생성하여 AI 학습·검증·보안 대응에 활용하는 데이터 자산

### AI 평가 데이터

AI 모델의 성능·정확성·안전성·업무 적합성을 검증하기 위한 평가용 데이터셋

### 데이터 파이프라인 (Feedback Loop)

사용자 반응과 모델 결과를 반영하여 데이터를 지속적으로 개선하는 순환형 데이터 운영 구조

---

# Governed

### 데이터 운영관리

데이터 공급·배포·활용 전 과정을 표준화하고 안정적으로 운영하기 위한 관리 체계

### 데이터 생명주기 관리

데이터 생성부터 보관·활용·폐기까지 전 과정을 정책 기반으로 관리하는 체계

### 데이터 디지털화

문서·이미지·오프라인 정보 등 비정형 자산을 활용 가능한 디지털 데이터로 전환하는 과정

### 데이터 보안 개인·민감정보 마스킹

개인정보와 민감정보를 보호하기 위해 접근 통제·암호화·마스킹을 적용하는 보안 관리 체계
