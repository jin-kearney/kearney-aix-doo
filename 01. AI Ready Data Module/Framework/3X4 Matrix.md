# Matrix에 넣을 때의 배치 기준

|단계|배치 기준|
|---|---|
|**수집**|어떤 데이터가 어디서 발생하고, 어떤 자산으로 확보되어야 하는가|
|**관리**|그 데이터를 신뢰 가능하고, 통제 가능하고, 연결 가능한 상태로 유지하려면 무엇이 필요한가|
|**활용**|AI 과제에서 실제 판단·검색·추천·자동화·최적화에 쓰이게 하려면 무엇이 필요한가|
