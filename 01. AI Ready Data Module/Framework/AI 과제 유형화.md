## AI 활용 주체 - 어디에 쓰이는가? (Where? Who?)


**1. 업무 인사이트형 AI**  <br>Business / Digital × Insight|
고객 데이터, CRM, 영업 이력, VOC, 상담 로그, 문서, 계약서, 재무·경영 KPI, ERP 데이터, 이메일·메신저 데이터|
데이터 통합, 자연어 처리, 분류·예측 모델, 추천 알고리즘, 이상 탐지, BI / 대시보드 연계, 설명 가능한 AI|**“업무 데이터 안에서 어떤 신호를 발견하고, 어떤 의사결정을 더 잘하게 만들 수 있는가?”**|

**2. 업무 실행형 AI**  <br>Business / Digital × Action|문서, 이메일, 보고서, 제안서, 회의록, 고객 문의, 업무 매뉴얼, 승인 요청, 청구서, 발주서, 지식 DB|생성형 AI, LLM, RAG, 워크플로우 자동화, RPA, API 연계, 권한 관리, 휴먼 인 더 루프, 출력 품질 검증
**“사람이 반복적으로 만들고 처리하는 디지털 업무 중 AI가 대신 수행할 수 있는 것은 무엇인가?”**

**3. 현장 인사이트형 AI**  <br>Physical / Industrial × Insight|센서 데이터, 설비 로그, 진동·온도·압력 데이터, 이미지·영상, 품질 검사 데이터, 공정 조건, 생산 실적, 에너지 사용량, 작업자·현장 데이터|컴퓨터 비전, 시계열 분석, 이상 탐지, 예지보전 모델, 품질 예측, Edge AI, 실시간 모니터링, OT 데이터 연계
**“현장 데이터에서 어떤 이상·위험·품질 문제·성능 저하를 선제적으로 감지할 수 있는가?”**

**4. 현장 실행형 AI**  <br>Physical / Industrial × Action|설비 제어 데이터, 로봇 동작 데이터, 공정 제어값, MES / SCADA / PLC 데이터, AGV / AMR 경로 데이터, 장비 상태 데이터, 디지털 트윈 데이터|로봇 제어, 공정 자동 제어, 폐루프 제어, 실시간 의사결정, Edge Computing, OT / IT 통합, 안전 제어, Fail-safe 설계
**“AI 판단을 실제 설비·공정·장비의 자동 실행으로 어떻게 연결할 것인가?”**
